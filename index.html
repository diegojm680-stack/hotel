Habitaciones

Contamos con diferentes tipos de habitaciones pensadas para brindarte comodidad, espacio y una estancia agradable, adaptándose a lo que necesitas.

Habitación pequeña

Diseñada para personas que viajan solas, esta habitación ofrece un espacio funcional con accesibilidad básica. Es ideal para personas con discapacidad visual o cognitiva, ya que cuenta con una distribución simple y fácil de comprender.

  • Espacios libres de obstáculos
  • Iluminación adecuada para baja visión
  • Señalización clara
  • Compatible con lector de pantalla
  • Acceso sencillo a mobiliario

Habitación mediana

Pensada para parejas o familias pequeñas, esta habitación incluye adaptaciones para personas con movilidad limitada o discapacidad motriz, permitiendo mayor comodidad en su desplazamiento.

  • Espacio amplio para movilidad
  • Acceso cómodo para silla de ruedas
  • Baño con fácil acceso
  • Controles accesibles
  • Diseño ergonómico
